# 這個檔案用來設置你想關注的新聞關鍵詞。
# 系統會自動抓取包含這些關鍵詞的熱榜新聞推送給你。
#
# 檔案分為兩個區域：
#   [GLOBAL_FILTER]  - 全局過濾區：排除不想看的內容
#   [WORD_GROUPS]    - 詞組定義區：設置想關注的關鍵詞
#
# ═══════════════════════════════════════════════════════════════

#                        全局過濾區
# ───────────────────────────────────────────────────────────────
# 在這裡寫入你不想看到的詞，每行一個。
# 包含這些詞的新聞會被自動排除，不會出現在推送中。
#
# 使用方法：
#   震驚              直接寫詞，包含"震驚"的新聞會被過濾
#   /賭博|博彩/       用 /.../ 包裹可以匹配多個詞（用 | 分隔）

[GLOBAL_FILTER]
# 過濾標題黨
震驚



# ───────────────────────────────────────────────────────────────
#                        詞組定義區
# ───────────────────────────────────────────────────────────────
# 在這裡寫入你想關注的關鍵詞。
# 每個詞組用空行分隔，同一詞組內的關鍵詞是"或"的關係。
#
# ┌─────────────────────────────────────────────────────────────┐
# │                      語法總覽（快速參考）                      │
# └─────────────────────────────────────────────────────────────┘
#
# 關鍵詞語法：
#   關鍵詞            普通關鍵詞，標題包含即匹配
#   /正則/            正則表達式匹配（自動忽略大小寫）
#   關鍵詞 => 別名    給關鍵詞指定顯示別名
#   [組別名]          詞組第一行，給整組指定別名
#   +關鍵詞           必須詞，所有必須詞都要匹配才算匹配
#   !關鍵詞           過濾詞，匹配則排除該條新聞（僅限當前詞組）
#   @數字             限制該詞組最多顯示多少條
#
# 顯示名稱優先級：
#   1. 有組別名 → 顯示組別名
#   2. 沒有組別名 → 顯示各行別名拼接（用 " / " 連接）
#   3. 沒有別名 → 顯示關鍵詞本身
#
#
# ┌─────────────────────────────────────────────────────────────┐
# │                      基礎用法（推薦新手）                      │
# └─────────────────────────────────────────────────────────────┘
#
# 1. 最簡單：直接寫關鍵詞
#    ────────────────────
#    華為
#
#    效果：匹配所有包含"華為"的新聞
#
#
# 2. 多個關鍵詞歸為一組
#    ────────────────────
#    華為
#    鴻蒙
#    任正非
#
#    效果：匹配包含"華為"或"鴻蒙"或"任正非"的新聞，統一顯示為"華為 / 鴻蒙 / 任正非"
#
#
# 3. 給詞組起個名字（推薦）
#    ────────────────────
#    [華為]
#    華為
#    鴻蒙
#    任正非
#
#    效果：同上，但顯示名稱為"華為"（更簡潔）
#
#
# ┌─────────────────────────────────────────────────────────────┐
# │                      進階用法（可選）                         │
# └─────────────────────────────────────────────────────────────┘
#
# 4. 用正則表達式匹配多個詞（一行搞定）
#    ────────────────────
#    /華為|鴻蒙|任正非/ => 華為
#
#    效果：匹配包含"華為"或"鴻蒙"或"任正非"的新聞，顯示為"華為"
#    說明：/.../ 裡用 | 分隔多個詞，=> 後面是顯示名稱
#
#    💡 不懂正則？問 AI：
#       "幫我寫一個正則表達式，匹配包含'華為'或'鴻蒙'或'任正非'的文本，
#        格式要求：/正則/ => 顯示名稱"
#
#
# 5. 精確匹配英文單詞（避免誤匹配）
#    ────────────────────
#    /\bAI\b/i => AI
#
#    說明：\b 表示單詞邊界，避免匹配到 "MAIL" 中的 "AI"
#          /i 表示忽略大小寫，"ai"、"AI"、"Ai" 都能匹配
#
#    💡 不懂正則？問 AI：
#       "幫我寫一個正則表達式，精確匹配英文單詞'AI'（不匹配 MAIL 中的 AI），
#        忽略大小寫，格式要求：/正則/i => 顯示名稱"
#
#
# 6. 排除特定內容
#    ────────────────────
#    [蘋果公司]
#    蘋果
#    !水果
#    !果園
#
#    效果：匹配"蘋果"但排除包含"水果"或"果園"的新聞
#    說明：! 開頭的詞表示"排除"
#
#
# 7. 限制顯示條數
#    ────────────────────
#    [科技新聞]
#    科技
#    @5
#
#    效果：最多顯示 5 條匹配的新聞
#    說明：@數字 表示限制條數
#
#
# 8. 必須同時包含多個詞
#    ────────────────────
#    +蘋果
#    +發布會
#
#    效果：必須同時包含"蘋果"和"發布會"才匹配
#    說明：+ 開頭的詞表示"必須包含"
#
# ───────────────────────────────────────────────────────────────

[WORD_GROUPS]
